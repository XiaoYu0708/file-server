# File Server

Windows HTTP 檔案伺服器，手機透過瀏覽器即可上傳/下載檔案。不需安裝任何 App，iPhone 開啟 Safari 即可操作。

## 使用方式

### 1. 啟動伺服器

```cmd
雙擊 dist/file_server.exe
```

第一次執行會跳出**檔案總管視窗**，選擇要分享的資料夾：

```
┌─ 選擇共享目錄 ──────────────────────┐
│  請選擇一個資料夾                    │
│  ┌──────────────────────────────┐   │
│  │ 📁 桌面                      │   │
│  │ 📁 文件                      │   │
│  │ 📁 下載                      │   │
│  │ 📁 D:\分享資料               │   │
│  └──────────────────────────────┘   │
│          [確定]                      │
└──────────────────────────────────────┘
```

之後每次啟動都會重新選擇（不存設定檔）。

### 3. 手機連線

| 方式 | 步驟 |
|------|------|
| **掃 QR Code** | iPhone 相機掃描終端機的 QR Code，自動開啟 Safari |
| **手動輸入** | Safari 輸入 `http://192.168.0.2:5000` |

## 功能一覽

| 功能 | 說明 |
|------|------|
| **上傳檔案** | 點擊上傳區選取檔案，支援拖曳 |
| **上傳資料夾** | 切換至「資料夾」模式，一次上傳整個目錄 |
| **下載檔案** | 點擊「下載」按鈕直接下載 |
| **下載資料夾** | 自動壓縮為 ZIP 封裝下載 |
| **刪除** | 點擊「刪除」按鈕（資料夾會連同內容一併刪除） |
| **鎖定** | 點擊右上角「鎖定」按鈕即時登出 |
| **QR Code** | 終端機顯示 QR Code，手機相機掃碼即開 |
| **Port 衝突處理** | 若 5000 埠被占用，自動搜尋 5000~5100 可用埠 |
| **安全性** | 每次啟動隨機產生 6 位數配對碼，關閉即失效 |

## 認證機制

```
┌─ 伺服器終端機 ───┐     ┌─ iPhone Safari ────┐
│  配對碼: 842916   │     │  輸入配對碼        │
│  （僅此處顯示）   │ ──→ │  842916            │
└───────────────────┘     │  [驗證通過]        │
                           │  → 顯示檔案列表    │
                           └────────────────────┘
```

- 配對碼僅顯示在**伺服器終端機**（需實體操作電腦才能看到）
- 所有 API 請求需帶 `X-Pair-Code` Header 或 URL 參數 `?code=`
- 關閉瀏覽器分頁或按「鎖定」即登出，重新整理頁面需重新輸入
- 伺服器關閉後配對碼即失效

## Port 衝突處理

若 Port 5000 已被其他程式占用，伺服器會自動搜尋 5000~5100 之間第一個可用埠：

```
⚠️  Port 5000 已被占用，自動切換至 Port 5001
```

QR Code、Web 連結、API 路徑都會自動對應到正確的 Port。

## 網路需求

- iPhone 與 Windows 需連接**同一台路由器**（同一個區域網路）
- Windows 防火牆需允許 TCP 連線（首次執行時請允許權限）
- 若連線逾時，請檢查分享器是否有啟用「AP 隔離」功能

## 從原始碼建置

需要 Docker Desktop。

```powershell
.\build.ps1
```

或手動建置：

```powershell
docker build -t file-server-builder .
$id = docker create file-server-builder
docker cp ${id}:/src/dist/file_server.exe dist/
docker rm $id
```

## 專案結構

```
file_server/
├── dist/file_server.exe     ← 執行檔（直接執行）
├── src/
│   ├── server.py            ← Flask 主程式（含 API 與 Web UI）
│   └── templates/index.html ← Web 介面（單頁 HTML）
├── Dockerfile               ← Wine + PyInstaller 交叉編譯
├── build.ps1                ← 一鍵建置腳本
└── requirements.txt         ← Python 依賴
```

## 技術棧

- **後端**: Python Flask → PyInstaller 打包為單一 .exe
- **前端**: 原生 HTML/CSS/JavaScript，無第三方框架
- **建置**: Docker + Wine 交叉編譯 Windows 執行檔
- **認證**: 隨機配對碼（支援 `X-Pair-Code` Header 與 URL 參數）
- **壓縮**: Python zipfile 模組（資料夾下載）
